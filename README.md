Custom V2Ray Routing Rules

This repository provides lightweight geoip.dat and geosite.dat routing rules. The files are automatically generated and optimized to fit within strict memory limits, specifically under 50MB, required by mobile clients like Happ on iOS.

Included categories:
- geosite: ru-available-only-inside, win-spy, category-ads-all
- geoip: yandex
